# administrador-de-productos-servidor

Proyecto que implementa una **REST API** utilizando diferentes tecnologías del ecosistema de JavaScript/TypeScript:

- **Node.js**: entorno de ejecución para JavaScript en el servidor.
- **Express**: framework minimalista para crear servidores y manejar rutas de forma sencilla y flexible.
- **Sequelize**: ORM que facilita la interacción con bases de datos SQL mediante modelos y consultas en JavaScript/TypeScript.
- **TypeScript**: superconjunto de JavaScript que añade tipado estático, mejorando la mantenibilidad y la detección temprana de errores.
